The First Half

In the first half we're going to focus on how to deal with information.

This includes:

  • Considering and defining requirements
  • Rehashing some cognition and perception
  • Looking at context and content
  • Information architecting
  • Wireframing and layout

The Second Half

In the second half, we're going to get you coding and delivering information effectively.

This includes:

  • Web coding basics (CSS/HTML)
  • Working with interaction
  • Responsive design
  • User experience
