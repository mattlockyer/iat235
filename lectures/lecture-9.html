Selectors - Relationships

  • A E - any child or child of a child all the way down (nested)
  • A > E - any child (direct descendant only)
  • E:first-child - the first child of the parent
  • A + E - the next sibling of A only
  • A ~ E - all siblings following A

You can get really specific, but you should do so in a sensible way...

Specifying a Font

One CSS property with many different formats... probably should do some reading on these!

Global Styles

font: inherit;

font: initial;

font: unset;

Get font from parent (inherit), from default (initial), or default if not specified in parent (unset)

These keywords can be applied to any CSS property!
