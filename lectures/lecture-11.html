Javascript Background

Javascript History

JavaScript was invented by Brendan Eich in a relatively short period of time to provide a programming interface to handle user interactions with elements in the DOM.

Events such as click and mouseup were provided as listeners, allowing coders to attach functions to these handle these events.

Since this basic implementation, JavaScript has evolved to fascilitate object oriented, full application, server side and hardware programming.

Why Javascript?

JavaScript is incredibly flexible, has no types and fail gracefully.

While these points can be incredibly powerful, they can also make debugging and building large JavaScript applications difficult.

We won't get into all the ins and outs of JavaScript and we'll stick to the basics of Brendan's original implementation, so debugging your scripts shouldn't become too difficult.

Selecting Elements

Notice how selecting elements using querySelector and querySelectorAll use the same syntax as CSS selectors?

FACT: those two JavaScript functions return exactly what you would expect if you use CSS Selectors.

querySelector will return the first element matching the selector in the DOM and querySelectorAll will return all elements matching the selector in the DOM.
